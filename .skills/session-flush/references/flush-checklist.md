# Flush checklist

## Save if present
- important decision
- meaningful next step
- blocker worth remembering
- preference or correction likely to matter again
- new business/brand/product context
- created or changed skill/doc that future work depends on

## Update these files selectively
- `memory/YYYY-MM-DD.md`
- `MEMORY.md`
- `PRIORITIES.md`
- `TASKS.md`
- relevant `cerebro/*` file
- relevant `.skills/*` file

## Skip if it is only
- chatter
- repetition
- obvious context already captured elsewhere
- low-value commentary
