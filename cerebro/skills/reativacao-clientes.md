# Skill - Reativação de Clientes

## Objetivo
Recuperar clientes que pararam de comprar.

## Lógica
1. identificar clientes sem compra dentro de uma janela definida
2. separar por tempo de inatividade
3. sugerir mensagem adequada por segmento
4. organizar lista de execução
5. registrar resultado para aprender o que converte

## Uso prático
Esta skill deve servir como base para campanhas simples e recorrentes de recuperação de vendas.
