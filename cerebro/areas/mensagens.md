# Mensagens - Família do Gás

## Objetivo
Criar mensagens simples, humanas e comerciais para recompra, reativação e atendimento.

## Tipos principais
- lembrete de recompra
- reativação de cliente parado
- resposta rápida de atendimento
- confirmação de pedido
- acompanhamento de cliente que não fechou

## Regras de mensagem
- ser curta
- ser clara
- parecer humana
- facilitar resposta
- puxar para ação

## Estrutura recomendada
1. saudação simples
2. contexto direto
3. oferta ou lembrança
4. chamada para resposta

## Exemplo de lógica
- cliente recorrente → lembrete amigável
- cliente parado → reativação com retomada leve
- cliente em atendimento → condução para fechamento

## Objetivo final
Melhorar conversão sem parecer mensagem fria ou robótica.
