# Dados de Clientes - Família do Gás

## Objetivo
Organizar os dados mínimos para aumentar recompra, reativação e eficiência comercial.

## Campos mínimos
- nome do cliente
- telefone
- endereço ou bairro
- data da última compra
- frequência aproximada de compra
- status do cliente
- observações úteis

## Status sugeridos
- ativo
- recorrente
- ocasional
- em risco
- inativo
- reativado

## Perguntas que os dados devem responder
- quem deve receber lembrete agora?
- quem está sumido?
- quem compra com frequência?
- quais regiões compram mais?
- quem vale priorizar primeiro?

## Regra prática
Sem histórico mínimo, não existe operação inteligente de retenção.
